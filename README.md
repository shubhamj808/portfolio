# Shubham Jadhav - Personal Portfolio Website

## Description

This is a modern, interactive personal portfolio website built with React and Three.js. It showcases my skills, experience, and projects in a visually appealing 3D environment. The website features responsive design, smooth animations, and interactive 3D elements.

## Live Demo

Visit the live website: [Shubham Jadhav Portfolio](https://shubhamj808.github.io/portfolio)

## Technologies Used

- **React.js** - Frontend library for building user interfaces
- **Three.js** - JavaScript 3D library
- **React Three Fiber** - React renderer for Three.js
- **Framer Motion** - Animation library for React
- **Tailwind CSS** - Utility-first CSS framework
- **EmailJS** - Client-side email sending
- **Vite** - Frontend build tool
- **React Router** - Routing library for React

## Features

- Interactive 3D elements and animations
- Responsive design for all device sizes
- Dark theme with space-inspired visuals
- Sections for:
  - Hero/Introduction
  - About Me
  - Experience Timeline
  - Skills Showcase
  - Contact Form
- 3D models and effects:
  - Stars background
  - 3D Earth model
  - Computer model
  - Interactive 3D balls for skills

## Installation

To run this project locally:

1. Clone the repository

   ```bash
   git clone https://github.com/shubhamj808/portfolio.git
   cd portfolio
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the development server

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Usage

Navigate through the different sections using the navigation bar. Interact with the 3D elements by moving your cursor over them. Fill out the contact form to send me a message directly.

## Deployment

This project is configured for deployment to GitHub Pages:

1. Update the homepage in `package.json` if needed
2. Run the deployment command
   ```bash
   npm run deploy
   ```

## Building for Production

To build the project for production:

```bash
npm run build
```

The built files will be in the `dist` directory.
